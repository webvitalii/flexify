# Flexify

Flexify is a frontend framework.

## Usage

```html
<section class="fx-g fx-u-bg-default">
	<div class="fx-g__box fx-g__box--12 fx-u-bg-primary">
		Header
	</div>
	<div class="fx-g__box fx-g__box--8 fx-u-bg-success">
		Content
	</div>
	<div class="fx-g__box fx-g__box--4 fx-u-bg-warning">
		Sidebar
	</div>
	<div class="fx-g__box fx-g__box--12 fx-u-bg-danger">
		Footer
	</div>
</section>
```
