# Flexify

Flexify is a frontend framework.

## Responsive grid

```html
    <div class="fx-g__box fx-g__box--12">
        Header
    </div>
    <div class="fx-g__box fx-g__box--12 fx-g__box--md-8">
        Content
    </div>
    <div class="fx-g__box fx-g__box--12 fx-g__box--md-4">
        Sidebar
    </div>
    <div class="fx-g__box fx-g__box--12">
        Footer
    </div>
</section>
```
